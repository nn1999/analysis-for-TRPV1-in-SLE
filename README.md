# analysis-for-TRPV1-in-SLE


This repository contains the analysis code associated with the study:

**TRPV1 restrains lupus pathogenesis by epigenetically restoring the MAFK-CMTM4 mechanical axis**

The scripts provided here were used to perform the computational and statistical analyses described in the manuscript. They are intended to support reproducibility and provide transparency for the analytical workflow used in the study.

For a detailed description of the experimental design, data processing procedures, statistical analyses, software versions, and parameter settings, please refer to the **Methods** section of the article.

## Notes

* This repository contains analysis code only.
* Any required input data, preprocessing steps, and interpretation of outputs should be understood in the context of the original manuscript.
